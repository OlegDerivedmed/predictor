## Key Notes

- As of Claude Code v2.1.3, slash commands and Agent Skills were **unified**: any `.md` file in `.claude/commands/` is now both a skill and a slash command.
- **Best starting points:** official `anthropics/skills` repo + community `hesreallyhim/awesome-claude-code` list.
- Skills live in `~/.claude/commands/` (global) or `.claude/commands/` (project-level).
